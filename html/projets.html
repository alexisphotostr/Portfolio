Hack the box


Hack The Box (HTB) is an online platform that offers virtual labs for practicing cybersecurity and penetration testing skills. It provides a range of real-world challenges, from beginner to expert, where users can test their hacking abilities in a safe and legal environment. HTB includes vulnerable machines and challenges that simulate security weaknesses, helping users to improve their problem-solving and ethical hacking skills.

SealTalk: my personnal project


I recently got hold of a broken laptop, took it apart, and repaired it. I installed Debian on it and set it up as a server. So far, I've only configured SSH, but I plan to turn it into a web server. I've developed an instant messaging app called SealTalk using Node.js for the backend and React.js for the frontend, and I'd like to host it on this server.
